Toolbox

Some of the main tools I work with:

Camtasia

Camtasia is a screen recorder and video editor software suite created and published by TechSmith. It is used to create and record video tutorials and presentations via screencast, or via a direct recording plug-in to Microsoft PowerPoint.

Articulate Storyline

Articulate Storyline is a software application used to create interactive e-learning courses. It is a popular choice for businesses and organizations of all sizes because it is easy to use and offers a wide range of features.

Vyond

Vyond is a cloud-based animated video creation platform that allows users of all skill levels to create professional-looking animated videos. It is a drag-and-drop interface that makes it easy to create videos without any prior video editing experience.

Github

GitHub is a cloud-based hosting service that helps developers store and manage their code, track and control changes to their code, and collaborate on projects with other developers. It is a popular platform for both open-source and commercial software development.
